graph-visualisation-0.1.0.2: Provides a series of graph-drawing functions for use with the algebraic-graphs library.

This library works with the algebraic-graphs library to visualise graphs. Different visualisation methods are included, each in their own module. However, the top-level module Visualise provides the main drawing functions drawGraph and drawGraph' along with re-exporting funtions from the Visualise.GraphViz module (which uses the renowned GraphViz library to help visualise graphs and therefore requires more specialised functions) and the Visualise.ExpressionTree module (whos customisable function requires extra parameters compared to the other modules). There a common functions which more than one of the modules rely on and these are exported by the Visualise.Common module.
