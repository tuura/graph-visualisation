The resultant graph drawing in an IO wrapper.

Uses Diagrams.TwoD.GraphViz as an interface to the Data.GraphViz library to produce a Diagrams representation of the provided graph using the provided GraphvizCommand. The resultant Diagram is wrapped in an IO from System.IO so must be bound to a function such as saveSVG to write it to a file. Uses the arrow paths produced by GraphViz but uses the custom functon draw to choose how to draw nodes. To draw graphs of graphs the Visualise.Tree module is used for the subgraph layouts.
